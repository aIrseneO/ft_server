Welcome to ft_server!

If you see this page, the nginx web server, WordPress service and phpMyAdmin service are successfully installed and working.

WordPress webpage is available at localhost/wordpress.com.
PhpMyAdmin is available at localhost/phpmyadmin.com.

For testing purpusing, MySQL infos are:
Database user : WP_DB_USER
Database password : WP_DB_PASSWORD

Further configuration setting can be found in the file install.sh located in the directory src.

Thank you for trying it out.
