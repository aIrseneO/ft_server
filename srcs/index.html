Welcome to ft_server!

If you see this page, the nginx web server, WordPress service and phpMyAdmin service are successfully installed and working.

Websites served by this server are:

  • WordPress.com: https://localhost:[Port]/wordpress.com
  • PhpMyAdmin.com: https://localhost:[Port]/phpmyadmin.com
  • Crules.com: https://localhost:[Port]/Crules.com
where [Port] is the port mapped to the container.

For testing, MySQL user and password created during the setup are:
User________: wpdbuser
Password____: wpdbpassword

Further configuration setting can be found in the file launch_server.sh located in the directory src.

Thank's for trying it out.
